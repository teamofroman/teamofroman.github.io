Проект: Telegram фреймворк

Начат 2024-01-12
Статус: в разработке
Репозиторий  
Простой фреймворк для работы с Telegram посредством Telegram Bot API

Демонстрирует работу с Telegram Bot API

Реализовано

  • базовый класс Bot: отправка запроса на сервер, цикл периодического опроса Telegram на наличие новых сообщений

  • функция разборки поступивших сообщений. Пока определяется только наличие новых сообщений и измененных сообщений

Технологии:

  • Python 3.10

  • Асинхронный http

  • poetry
