Проект: Список дел и задач

Начат 2023-08-05
Статус: в разработке
Репозиторий  
Проект предоставляет возможность вести список дел и задач.

Составление списка и отслеживание дел и задач. Доступен только для авторизированных пользователей.

Реализовано

  • отображение списка дел и задач по категориям Просроченные, Со сроком, Безсрочные, Завершенные

  • показывается статистика задач по категориям

  • реализовано создание задачи, ее редактирование и удаление

Технологии

  • Python 3.10

  • Django 4.2

  • Bootstarp

  • SQLite
